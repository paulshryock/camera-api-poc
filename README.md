# Camera API POC

Quick proof-of-concept for turning the device camera on and off.

## Instructions

1. Clone this repository
2. Run `npm install`
3. Run `npm start`
4. Visit `localhost:5000` in a browser

## Notes

There is no "build step". Just serve these static files on a CDN:

- `index.html`
- `main.css`
- `main.js`
